# **Personal Portfolio Website**

A modern and responsive personal portfolio website built using **React**, **Vite**, **JavaScript**, **HTML**, **CSS**, and **Tailwind CSS**. This project showcases my skills, projects, experience, and contact information in a clean and interactive user interface.

## 🚀 **Features**

- **Responsive design** for desktop, tablet, and mobile devices
- **Modern and clean user interface**
- **Fast development and build process** using Vite
- **Component-based architecture** with React
- **Styled with Tailwind CSS** for rapid and consistent UI development
- **Project showcase section**
- **About Me section**
- **Skills and Technologies section**
- **Contact section**
- **Smooth navigation and user experience**

## 🛠️ **Technologies Used**

- **React** – Frontend JavaScript library
- **Vite** – Fast build tool and development server
- **JavaScript (ES6+)** – Programming language
- **HTML5** – Markup structure
- **CSS3** – Additional styling
- **Tailwind CSS** – Utility-first CSS framework
